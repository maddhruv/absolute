# absolute-docs

Diátaxis-driven documentation. Write, improve, or audit tutorials, how-tos, reference, explanation, and developer docs. Gates on the outline before writing prose; verifies every claim against the code.

## Quick Facts

| Field | Value |
|-------|-------|
| Category | workflow |
| Version | 0.5.0 |
| Platforms | claude-code, gemini-cli, openai-codex, mcp |
| License | MIT |
| Deep-dive guides | `references/` (developer-docs, docs-stacks, explanation, how-to-guides, reference, style-and-voice, tutorials) |

## How to Install

```bash
npx skills add maddhruv/absolute-docs
```

## Usage

```bash
/absolute-docs "Write a getting-started tutorial for the CLI"
/absolute-docs "Audit our docs site and propose a restructure"
/absolute-docs "Write a README for this package"
```
